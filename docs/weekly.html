朝の新顔枠 score 55.11 登場 1回 stars 1051 Rust

選定理由: Rust製の高速TS/JSコードベース解析、静的・動的の

Codebase intelligence for TypeScript and JavaScript. Free static layer: unused code, duplication, circular deps, complexity hotspots, arc...

FallowはTypeScript/JavaScriptコードベース全体を解析するツール。
ファイル単位ではなくプロジェクト全体の依存関係グラフから、
未使用コード、重複、循環依存、複雑性のホットスポットを検出する。

Rustで書かれていてサブ秒で動作。90以上のフレームワークに対応し、
オプションで本番トラフィックに基づく実行パス分析も可能。
AIが生成したコードの品質確認や、コードベースの健全性維持に役立つ。

静的解析は無料でOSS。CLI、LSP、MCPサーバーとしても使える。
https://github.com/fallow-rs/fallow
